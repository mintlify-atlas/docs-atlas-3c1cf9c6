## Welcome to Template API Rust

Template API Rust is a production-ready foundation for building fast, scalable REST APIs in Rust. Built with modern async patterns and type-safe database access, this template eliminates boilerplate while providing a solid architectural foundation for your API services.

## Technology Stack

- **Rust 1.83** - Modern, safe systems programming
- **Axum 0.7** - Ergonomic web framework built on Tower and Hyper
- **Tokio 1.37** - High-performance async runtime
- **SQLx 0.6** - Async, pure-Rust SQL toolkit with compile-time query checking
- **Serde** - Serialization framework for JSON handling
- **Tracing** - Application-level tracing for diagnostics

## Use Cases

This template is ideal for:

- **Microservices** - Build lightweight, focused API services
- **Backend APIs** - Power web and mobile applications
- **Data Processing** - High-throughput data ingestion and transformation
- **Prototyping** - Quickly validate API designs with production-quality code

## What's Included

The template provides a working example with a City resource demonstrating:

- Full CRUD operations (Create, Read, Update, Delete)
- Query filtering with multiple operators (equality, case-insensitive pattern matching)
- Pagination support
- JSON request/response handling
- Structured error responses
- Database connection pooling
- Environment-based configuration
