Template API Rust implements a generic repository pattern using Rust traits. This allows you to define database operations once and reuse them across all models.

## Overview

The repository pattern provides four core traits:

- **`Insertable`** - Create new records
- **`Updatable`** - Update existing records
- **`Deletable`** - Delete records by ID
- **`Listable`** - Query and filter records with pagination

## The Insertable Trait

Defines how to insert a model into the database:

```rust src/repositories/insertable.rs
use sqlx::{Postgres, postgres::PgArguments};

pub trait Insertable {
    const TABLE: &'static str;
    const COLUMNS: &'static [&'static str];

    fn bind<'q>(
        self,
        query: sqlx::query::QueryAs<'q, Postgres, Self, PgArguments>,
    ) -> sqlx::query::QueryAs<'q, Postgres, Self, PgArguments>
    where
        Self: Sized;
}
```

### Example Implementation

```rust src/models/city.rs
impl Insertable for City {
    const TABLE: &'static str = "cities";
    const COLUMNS: &'static [&'static str] = &["name", "region"];

    fn bind<'q>(
        self,
        query: sqlx::query::QueryAs<'q, Postgres, Self, PgArguments>,
    ) -> sqlx::query::QueryAs<'q, Postgres, Self, PgArguments> {
        query
            .bind(self.name)
            .bind(self.region)
    }
}
```

<Note>
  The `id` column is excluded from `COLUMNS` because it's typically auto-generated by the database.
</Note>

## The Updatable Trait

Defines how to update existing records:

```rust src/repositories/updatable.rs
use sqlx::{Postgres, postgres::PgArguments};

pub trait Updatable {
    /// Table name
    const TABLE: &'static str;

    /// Columns to update (WITHOUT id)
    const COLUMNS: &'static [&'static str];

    /// WHERE column (e.g., "id")
    const ID_COLUMN: &'static str;

    /// Bind columns + id (AT THE END)
    fn bind_update<'q>(
        self,
        query: sqlx::query::QueryAs<'q, Postgres, Self, PgArguments>,
    ) -> sqlx::query::QueryAs<'q, Postgres, Self, PgArguments>
    where
        Self: Sized;
}
```

### Example Implementation

```rust src/models/city.rs
impl Updatable for City {
    const TABLE: &'static str = "cities";
    const COLUMNS: &'static [&'static str] = &["name", "region"];
    const ID_COLUMN: &'static str = "id";

    fn bind_update<'q>(
        self,
        query: sqlx::query::QueryAs<'q, Postgres, Self, PgArguments>,
    ) -> sqlx::query::QueryAs<'q, Postgres, Self, PgArguments> {
        query
            .bind(self.name)
            .bind(self.region)
            .bind(self.id) // WHERE id = $n
    }
}
```

<Warning>
  The ID must be bound **last** in `bind_update` to match the generated SQL WHERE clause.
</Warning>

## The Deletable Trait

Simplest trait - only requires table and ID column names:

```rust src/repositories/deletable.rs
pub trait Deletable {
    /// Table name
    const TABLE: &'static str;

    /// ID column (usually "id")
    const ID_COLUMN: &'static str;
}
```

### Example Implementation

```rust src/models/city.rs
impl Deletable for City {
    const TABLE: &'static str = "cities";
    const ID_COLUMN: &'static str = "id";
}
```

## The Listable Trait

Defines filtering, pagination, and querying:

```rust src/repositories/listable.rs
pub enum FilterOp {
    Eq(String),      // Exact match: column = value
    ILike(String),   // Case-insensitive like: column ILIKE '%value%'
}

pub trait Listable {
    const TABLE: &'static str;
    const COLUMNS: &'static [&'static str];

    /// Return filters as (column, operation) pairs
    fn filters(&self) -> Vec<(&'static str, FilterOp)>;

    fn page(&self) -> Option<u32> {
        None
    }

    fn per_page(&self) -> Option<u32> {
        None
    }
}
```

### Example Implementation

The `Listable` trait is implemented on **resource structs** (query parameters), not models:

```rust src/resources/city_resource.rs
use serde::Deserialize;

#[derive(Deserialize, Default)]
pub struct CityResource {
    pub name: Option<String>,
    pub region: Option<String>,
    pub page: Option<u32>,
    pub per_page: Option<u32>,
}
```

```rust src/models/city.rs
impl Listable for CityResource {
    const TABLE: &'static str = "cities";
    const COLUMNS: &'static [&'static str] = &["id", "name", "region"];

    fn filters(&self) -> Vec<(&'static str, FilterOp)> {
        let mut v = Vec::new();

        if let Some(name) = &self.name {
            v.push(("name", FilterOp::ILike(name.clone())));
        }

        if let Some(region) = &self.region {
            v.push(("region", FilterOp::Eq(region.clone())));
        }

        v
    }

    fn page(&self) -> Option<u32> {
        self.page
    }

    fn per_page(&self) -> Option<u32> {
        self.per_page
    }
}
```

<Tip>
  Use `FilterOp::ILike` for partial text matching and `FilterOp::Eq` for exact matches.
</Tip>

## Generic Service Functions

The traits enable generic service functions that work with any model:

### Generic Insert

```rust src/services/generic_query.rs
pub async fn generic_insert<T>(value: T, pool: &Pool<Postgres>) -> Result<T, Error>
where
    T: Insertable + for<'r> sqlx::FromRow<'r, sqlx::postgres::PgRow> + Send + Unpin,
{
    let columns = T::COLUMNS.join(", ");

    let placeholders = (1..=T::COLUMNS.len())
        .map(|i| format!("${}", i))
        .collect::<Vec<_>>()
        .join(", ");

    let query = format!(
        "INSERT INTO {} ({}) VALUES ({}) RETURNING *",
        T::TABLE,
        columns,
        placeholders
    );

    let q = sqlx::query_as::<_, T>(&query);
    let q = value.bind(q);
    q.fetch_one(pool).await
}
```

### Generic Update

```rust src/services/generic_query.rs
pub async fn generic_update<T>(value: T, pool: &Pool<Postgres>) -> Result<T, Error>
where
    T: Updatable + for<'r> FromRow<'r, PgRow> + Send + Unpin,
{
    let set_clause = T::COLUMNS
        .iter()
        .enumerate()
        .map(|(i, col)| format!("{} = ${}", col, i + 1))
        .collect::<Vec<_>>()
        .join(", ");

    let where_index = T::COLUMNS.len() + 1;

    let query = format!(
        "UPDATE {} SET {} WHERE {} = ${} RETURNING *",
        T::TABLE,
        set_clause,
        T::ID_COLUMN,
        where_index
    );

    let q = sqlx::query_as::<_, T>(&query);
    let q = value.bind_update(q);
    q.fetch_one(pool).await
}
```

### Generic Delete

```rust src/services/generic_query.rs
pub async fn generic_delete<T, ID>(id: ID, pool: &Pool<Postgres>) -> Result<T, Error>
where
    T: Deletable + for<'r> FromRow<'r, PgRow> + Send + Unpin,
    for<'q> ID: Send + Sync + sqlx::Encode<'q, Postgres> + sqlx::Type<Postgres>,
{
    let query = format!(
        "DELETE FROM {} WHERE {} = $1 RETURNING *",
        T::TABLE,
        T::ID_COLUMN
    );

    sqlx::query_as::<_, T>(query.as_str())
        .bind(id)
        .fetch_one(pool)
        .await
}
```

### Generic List

```rust src/services/generic_query.rs
pub async fn generic_list<T, F>(
    filter: F,
    order_by: Option<(&'static str, &'static str)>,
    pool: &Pool<Postgres>,
) -> Result<Vec<T>, Error>
where
    T: for<'r> FromRow<'r, PgRow> + Send + Unpin,
    F: Listable,
{
    let columns = F::COLUMNS.join(", ");
    let mut query = format!("SELECT {} FROM {}", columns, F::TABLE);

    let filters = filter.filters();
    let mut bind_index = 1;
    let mut binds = Vec::new();

    // Build WHERE clause
    if !filters.is_empty() {
        query.push_str(" WHERE ");

        for (i, (col, op)) in filters.iter().enumerate() {
            if i > 0 {
                query.push_str(" AND ");
            }

            match op {
                FilterOp::Eq(val) => {
                    query.push_str(&format!("{} = ${}", col, bind_index));
                    binds.push(val.clone());
                }
                FilterOp::ILike(val) => {
                    query.push_str(&format!("{} ILIKE ${}", col, bind_index));
                    binds.push(format!("%{}%", val));
                }
            }

            bind_index += 1;
        }
    }

    // Add ORDER BY
    if let Some((col, dir)) = order_by {
        let d = if dir.eq_ignore_ascii_case("DESC") { "DESC" } else { "ASC" };
        query.push_str(&format!(" ORDER BY {} {}", col, d));
    }

    // Add pagination
    if let (Some(page), Some(per_page)) = (filter.page(), filter.per_page()) {
        let offset = (page.saturating_sub(1) * per_page) as i64;
        query.push_str(&format!(" LIMIT {} OFFSET {}", per_page, offset));
    }

    let mut q = sqlx::query_as::<_, T>(&query).persistent(false);

    for v in binds {
        q = q.bind(v);
    }

    q.fetch_all(pool).await
}
```
